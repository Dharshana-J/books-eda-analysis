# Books EDA Analysis

## Project Overview

This project focuses on Exploratory Data Analysis (EDA) of a books dataset containing 200+ records.

The analysis was performed using Python, Pandas, and Matplotlib to understand the structure of the dataset, identify patterns, detect anomalies, and generate insights.

## Questions Explored

1. How many records are present in the dataset?
2. What are the data types of each column?
3. Are there any missing values?
4. Which rating category is most common?
5. What is the price distribution of books?
6. Which books are the most expensive?

## Tools Used

* Python
* Pandas
* Matplotlib

## Analysis Performed

* Dataset Structure Analysis
* Data Type Inspection
* Missing Value Analysis
* Rating Distribution Analysis
* Price Distribution Analysis
* Top Expensive Books Analysis

## Key Findings

* Dataset contains more than 200 book records.
* No significant missing values were found.
* Most books are concentrated within a moderate price range.
* A few books have relatively high prices and can be considered outliers.
* Ratings are distributed across multiple categories.
